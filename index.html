CSS Snippits!

Here's a quick reference sheet for several advanced CSS concepts.

Try adjusting the window size to see what it does to the cards!

Flexbox Row

These three properties create

a Flexbox row layout! 

    .row {
        display: flex;
        flex-direction: row;
        flex-wrap:wrap;
    }

Flexbox Column

These three properties create

a Flexbox column layout! 

    .row {
        display: flex;
        flex-direction: column;
        flex-wrap:wrap;
    }

CSS Grid Layout

Build a 12-column layout using CSS Grid! 

    .grid {
        display:grid;
        width: 100%;
        grid-template-columns: repeat (12, 1fr);
    }

Linear Gradients

This will create a linear gradient from top to bottom! 

    .linear-gradient-background {
        background-image: linear-gradient (
            rgb(97, 74, 109) 0%,
            rgb(50, 30, 65) 100%
        );
    }

Box Transition Glow

Shadows glow when you hover over

elements with your mouse! 

    .code-card .card-header {
        border-radius: 8px;
        transition: all 0.5s ease
    }
    .code-card:hover,
    .code-card:hover .card-header {
        box-shadow: inset 0px 0px 8px
    }

Overlay Card with Title

Positions the title above its relative starting point! 

    .card-header {
        position: relative;
        martin-top: -20px;
    }
